Gluu Server Backup#

There methods of backing up Gluu Server CE is given below:

  • Tarball Method: using tar command to tarball the entire Gluu Server CE chroot folder

Tarball Method#

  1. Stop the server using below command a. #service gluu-server-3.0.0 stop
  2. use tar command to take a back up b. # tar cvf gluu244-backup.tar /opt/gluu-server-3.0.0/
